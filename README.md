# Android APK Demo (GitHub Actions)

هذا مشروع Android بسيط (Java) هدفه الوحيد انك ترفعه على GitHub ويعمل Build APK تلقائياً عبر GitHub Actions.

## كيف تاخد الـ APK
1) ارفع المشروع على GitHub (على فرع `main`)
2) افتح تبويب **Actions** داخل الريبو
3) افتح آخر Run باسم **Build APK**
4) من قسم **Artifacts** حمّل `app-debug-apk`

> ملاحظة: هذا يبني Debug APK للتجربة. (Release يحتاج توقيع keystore)
